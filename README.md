# Meeting service

**Backend:** .NET Core  
**Database:** SQLite

**Open API:** Swagger 

## Start project in Development Mode

### Requirements
* .NET Core 3.1+

### Start the project
* Open .sln file with Visual Studio and start the project<br/>
or<br/>
* Open terminal in Meetings.API folder, enter `dotnet run` and navigate to [http://localhost:5000](http://localhost:5000)
